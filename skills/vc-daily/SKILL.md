# 投资日报 — VC/PE 内容采集与生成

## 内容模块

1. **今日融资速递**：当日已披露的融资事件（轮次、金额、投资机构、赛道）
2. **赛道热力图**：本周/本月融资最活跃的细分赛道
3. **机构动态**：知名 VC/PE 机构的最新出手、退出、基金动态
4. **趋势洞察**：基于融资数据的赛道趋势分析（1-2个）
5. **值得关注的公司**：本日融资中的亮点项目，附简介

---

## 采集策略

```
搜索关键词组合（每次搜索间隔2秒）：
1. web_search("融资 今日 {date} million")
2. web_search("startup funding today {date}")
3. web_search("VC investment news {date}")
4. web_search("创投 融资 {赛道关键词} {date}")
5. 对候选公司用 web_fetch 验证原始报道
```

**数据源优先级**：
- 36氪融资快讯、IT桔子（国内融资）
- TechCrunch、Crunchbase News（海外融资）
- 投中网、创业邦（行业分析）
- 彭博、路透（大额融资验证）

## 质量要求

- **融资金额和轮次必须来自可验证来源**，不猜测
- 未披露金额标注"金额未披露"
- 每笔融资附原始报道链接
- 趋势洞察基于数据，不做夸大预测
- 如当日融资新闻较少，诚实说明"今日公开融资信息有限"，不凑数
