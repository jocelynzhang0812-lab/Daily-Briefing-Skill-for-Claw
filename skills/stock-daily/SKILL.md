# 股票日报 — 自选股监控与日报生成

## 输入配置

从 `workspace/config/daily-briefing-config.txt` 读取：
```
自选股: 600519,000858,AAPL,NVDA   # A股用6位代码，美股用代号
关注板块: 新能源,AI芯片             # 可选，监控板块动态
```

若配置中无自选股，**必须先询问用户**：
> "请告诉我你的自选股代码（如 A股：600519，美股：AAPL），我来生成你的专属股票日报。"

---

## 内容模块

1. **大盘概况**：上证/深证/创业板/恒生/纳指涨跌，成交量
2. **自选股行情**：每只自选股的当日涨跌、价格、换手率
3. **个股新闻**：每只自选股的重要公告、新闻（过去24小时）
4. **板块动态**：用户关注板块的整体表现
5. **主力资金**：北向资金、两融数据（使用公开数据）
6. **明日关注**：次日需关注的重要事件（财报、解禁、政策等）

---

## 采集策略

```
逐步搜索：
1. web_search("上证指数 今日收盘 {date}")
2. for each 自选股:
   web_search("{股票名称} {股票代码} 今日行情 {date}")
   web_search("{股票名称} 最新公告 新闻 {date}")
3. web_search("北向资金 今日 {date}")
4. web_search("{关注板块} 板块 今日表现 {date}")
```

**数据源**：
- 行情数据：东方财富、同花顺、雪球（公开展示数据）
- 公告：上交所/深交所官网、巨潮资讯
- 财经新闻：财联社、证券时报、上海证券报

```
创业板指：XXXX.XX ▲+X.XX%

━━━ 自选股播报 ━━━

📌 [股票名称]（XXXXXX）
   现价：XX.XX  涨跌：▲+X.XX%  换手：X.XX%
   今日要闻：
   • [新闻标题](链接) — 一句话摘要
   • [公告标题](链接) — 一句话摘要

```

---

## 质量要求

- **行情数据标注数据时间**（盘中/盘后）
- 价格数据尽量来自可搜索的公开来源，无法确认时注明"数据待核实"
- **不给出买卖建议**，仅陈述事实
- 末尾必须加免责声明
- 若自选股数量超过10只，优先展示涨跌幅排名前5和后5
