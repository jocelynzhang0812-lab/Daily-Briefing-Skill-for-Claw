# 通用日报 — 内容采集与生成

## 内容模块

按以下模块采集当日内容（优先级从高到低）：

1. **头条要闻**：当日最重要的全球/国内大事（3-5条）
2. **金融市场**：A股/港股/美股涨跌概况、汇率、大宗商品（使用公开市场数据）
3. **科技 & AI**：科技公司动态、AI产品发布、重要技术突破（3-5条）
4. **中国政策**：重要政策发布、监管动态、产业政策（2-3条）
5. **国际动态**：地缘政治、国际关系重要事件（2-3条）

用户配置中的`关注`字段可调整各模块权重，关注领域的内容优先展示。

---

## 采集策略

```
搜索顺序：
1. web_search("[关注领域] 今日新闻 {date}"）
2. web_search("今日财经要闻 {date}")
3. web_search("科技AI最新动态 {date}")
4. 每个搜索结果用 web_fetch 验证原始来源
```

**数据源优先级**（从高到低）：
- 新华社、人民日报、央视新闻（政策类）
- 财联社、澎湃新闻、21世纪经济报道（财经类）
- 36氪、虎嗅、IT之家（科技类）
- 环球网、观察者网（国际类）

## 消息流格式输出结构

使用 Markdown，标题加粗，每条新闻附链接。分为5个段落逐一发送，便于阅读。

---

## 质量要求

- 每条新闻**必须附原始链接**
- 摘要不超过50字，客观陈述事实
- 不发表主观评论（除非用户配置要求"深度分析"）
- 市场数据注明来源和时间戳
