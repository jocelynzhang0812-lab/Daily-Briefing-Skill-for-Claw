# Daily Briefing Skill

> 面向 Kimi Claw 的个性化日报生成技能，支持通用新闻、VC投资、股票持仓、AI效率四种日报类型。

---

## 目录结构

```
daily-briefing/
├── SKILL.md                        # 主入口：配置管理、路由分发
├── skills/
│   ├── daily-core/SKILL.md        # 子技能：通用新闻日报
│   ├── vc-daily/SKILL.md          # 子技能：VC/PE 投资日报
│   ├── stock-daily/SKILL.md       # 子技能：股票持仓日报
│   └── ai-daily/SKILL.md          # 子技能：AI 效率日报（反炒作）
├── templates/                      # PDF HTML 模板（可自行扩展）
└── references/
    ├── data-sources.md             # 数据源参考手册 + 搜索关键词模板
    └── config-template.txt         # 用户配置文件模板
```

---

## 安装方式

1. 解压 `daily-briefing.zip`
2. 将 `daily-briefing/` 文件夹放入 Kimi Claw 的 skills 目录
3. 重启 Kimi Claw，技能自动加载

---

## 快速开始

直接对 Kimi Claw 说：

```
生成今日日报
帮我出一份 AI 效率日报
给我看看今天的投资动态
出一份股票日报
```

**首次使用**时，Claw 会引导你完成配置（日报类型、关注领域、输出格式）。配置保存在：

```
workspace/config/daily-briefing-config.txt
```

后续使用直接沿用配置，无需重复设置。

## 日报类型

### 通用日报
面向普通读者，覆盖地缘政治、金融市场、科技 AI、中国政策等多领域。

**触发示例**：`生成今日新闻摘要` / `帮我看看今天有什么重要的`

**输出内容**：
- 今日头条（3-5条）
- 金融市场概况（A股/港股/美股/汇率/大宗商品）
- 科技 & AI 动态
- 中国政策要闻
- 国际动态

---

### 投资日报（VC/PE）
面向投资人，聚焦融资事件与赛道趋势。

**触发示例**：`出一份投资日报` / `今天有什么融资新闻`

**输出内容**：
- 今日融资速递（轮次/金额/投资方/赛道）
- 赛道热力图
- 机构动态
- 趋势洞察

---

### 股票日报
面向股票投资者，监控自选股行情与个股新闻。

**触发示例**：`给我出股票日报` / `帮我看看我的自选股今天怎么样`

**输出内容**：
- 大盘概况（上证/深证/创业板/恒生/纳指）
- 自选股逐只播报（价格/涨跌/换手率/个股要闻）
- 板块动态
- 北向资金
- 明日关注事项

> ⚠️ 本日报仅供参考，不构成投资建议。

---

### AI 效率日报（反炒作）
面向开发者和超级个体，扫描 GitHub / HuggingFace / 官方博客等权威来源，筛选可立即行动的高价值更新。

**触发示例**：`出一份 AI 效率日报` / `今天 AI 圈有什么值得关注的`

**输出内容**：

| 等级 | 内容 | 数量上限 |
|------|------|----------|
| S级 核心关注 | 可验证 + 可立即行动的工具/技术突破 | 1-2 条（宁缺毋滥）|
| B级 工具箱更新 | 实用效率工具，有 Demo/代码 | 3-6 条 |
| C/D级 行业降噪 | 仅需知晓的背景信息 | 1-3 条 |
| 今日行动建议 | 3条可执行的最小动作 | 固定3条 |

**数据来源白名单**：`github.com` / `huggingface.co` / `openai.com` / `anthropic.com` / `ai.meta.com` / `ai.googleblog.com`

## 配置文件说明

配置文件路径：`workspace/config/daily-briefing-config.txt`

```
类型: 通用          # 通用 / 投资 / 股票 / AI效率
关注: 科技,金融,中国  # 关注领域（通用日报）
自选股: 600519,AAPL  # 股票代码（股票日报）
关注板块: 新能源,AI芯片
输出: 消息流         # PDF / 飞书文档 / 消息流
语言: 中文
语音: 关闭           # 开启 / 关闭
深度分析: 关闭        # 开启后增加分析视角
```

---

## 输出格式

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| 消息流 | Markdown 格式，直接在对话中展示 | 日常快速阅读 |
| PDF | HTML 模板渲染后导出，适合存档 | 正式报告/分享 |
| 飞书文档 | 转换为飞书文档块，生成文档链接 | 团队协作/订阅推送 |

## 扩展新日报类型

1. 在 `skills/` 下新建目录，如 `skills/research-daily/`
2. 编写 `SKILL.md`，定义内容采集逻辑和输出格式
3. 在配置文件中设置 `类型: 学术`
4. 在主 `SKILL.md` 的路由表中添加对应条目

---

## 设计原则

- **不编造数据**：所有内容来自实际搜索结果，网络受限时主动说明
- **来源可追溯**：每条新闻/资讯附原始链接
- **宁缺毋滥**：无可验证内容时留空，不以假充真
- **反炒作**：AI效率日报过滤营销号标题，只讲事实和价值
